In this activity, learn how to use inline, internal, external CSS and also learn about element, class, id and global selectors.

1. Element Selector

All <h1> tags are blue because of the h1 { ... } rule in internal CSS.

2. ID Selector

This card has id="super-important", so the #super-important { ... } rule makes it look extra special.

3. Class Selector

These boxes use class="css-card". The .css-card { ... } rule gives them borders, padding, and round corners. We can reuse this class on MANY elements.

4. Inline CSS

Inline CSS is written directly on the HTML tag using the style attribute.

5. External CSS

This page is also linked to an external file called styles.css. Try changing that file to see what happens!

I am styled using EXTERNAL CSS in styles.css.
